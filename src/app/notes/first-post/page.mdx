# Selamat Datang di Dunia Catatan Saya

Ini adalah konten *notes* pertama saya yang dibuat menggunakan **MDX** di Next.js.

MDX memungkinkan saya untuk menulis dengan Markdown biasa:

* **List Item 1:** Cepat dan mudah.
* **List Item 2:** Fokus pada konten.

Dan saya juga bisa menggunakan komponen React, misalnya tombol dari shadcn/ui:

<Button>Ini Komponen React</Button>
